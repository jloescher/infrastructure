How It Works

Encryption

Secrets are encrypted at rest using SOPS with age encryption. The private key is stored on the dashboard server.

Injection

During deployment, secrets are decrypted and injected into application environment variables or .env files.

Precedence

App-specific secrets override global secrets with the same key. Additional runtime variables can be set per deployment.
