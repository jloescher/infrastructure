Manual Deployment

Or deploy manually via SSH:

# Deploy to all servers
ssh root@100.92.26.38 "/opt/scripts/deploy-app.sh {{ app_name }}"
ssh root@100.89.130.19 "/opt/scripts/deploy-app.sh {{ app_name }}"

Deployment Model: PULL + Webhook

This application uses pull-based deployment:

  • Servers pull code directly from GitHub
  • No SSH credentials stored in GitHub
  • Secrets stored encrypted in dashboard (SOPS)
  • Deploy via dashboard or webhook

GitHub Webhook (recommended)

Configure a GitHub webhook to trigger automatic deployments over the public endpoint:

POST {{ webhook_base_url }}/{{ app_name }}
Content-Type: application/json
X-Hub-Signature-256: sha256=...

Event: push

Webhook secret (HMAC SHA-256):

{{ webhook_secret }}
